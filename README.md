# Microsoft Foundry Computing History Agent

An AI-powered Computing History Assistant built using Microsoft Foundry, Azure AI Projects SDK, OpenAI Responses API, and Flask.

## Features

* Microsoft Foundry Agent
* Azure Entra ID Authentication
* OpenAI Responses API
* Flask Web Interface
* Multi-turn Conversation Support
* Computing History Knowledge Base

## Technologies Used

* Python
* Flask
* Microsoft Foundry
* Azure AI
* OpenAI SDK
* Azure Identity

## Setup

```bash
pip install -r requirements.txt
az login
flask --app app run --port 5001
```

Open:

```
http://127.0.0.1:5001
```

## Example Questions

* What was ENIAC?
* How does it compare with COLOSSUS?
* Tell me about the Commodore 64.
* What was Grace Hopper's contribution to computing?
